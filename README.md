# Personal Portfolio

This is a personal portfolio showcasing skills, projects, achievements, and professional experience. It is designed to present work clearly and professionally.

---

## Features

- Responsive layout for desktop, tablet, and mobile  
- Sections for About, Projects, Skills, and Contact  
- Easy navigation with smooth scrolling  
- Links to social media and professional profiles  
- Project showcase with descriptions and external links  

---

## Technologies Used

- HTML, CSS, JavaScript   
- Can be hosted on GitHub Pages, Netlify, or Vercel
